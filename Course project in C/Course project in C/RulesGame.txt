+------------------------|Правила игры-головоломки|----------------------+
|   Для решения данной головолмки необходимо передвинуть каждую клетку с |
|числом в одном из четырех направлений, используя координаты ячеек (x,y),|
|число указывает количество клеток, на которое можно осуществить         |
|перемещение. Перемещая число, оно уменьшается на 1, до тех пор, пока оно|
|не станет равным 0, если число станет равно 0, то в клетку запишится    |
|символ X. Когда все перемещения выполнены, все пустые клетки должны быть|
|соединены друг с другом по горизонтали и вертикали, а клетки с символом |
|X не должны иметь общую сторону.                                        |
|   Необходимо соблюдать следующие правила:                              |
|   1. Перемещать клетки с числами за пределы игрового поля запрещено;   |
|   2. Перемещать клетки с числами в клетки, в которых уже имеется число |
|      или символ X, запрещено;                                          |
|   3. Перемещать пустые клетки запрещено;                               |
|   4. Перемещать символ X запрещено.                                    |
|   Соблюдая данные правила, Вы, безусловно, сможете решить головоломку и|
|достигнуть желаемой победы.                                             |
|   Если Вы ознакомилися с правилами, нажмите ENTER для возвращение в    |
|меню                                                                    |
